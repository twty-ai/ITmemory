# IT Memory — Transway TMS

Persistent knowledge base for the Transway TMS project. Auto-updated by Claude Code at the end of each working session.

## Structure

- `MEMORY.md` — Project-wide memory index (deployment, stack, gotchas)
- `communications.md` — Communications Hub architecture
- `trip_activities_mileage.md` — Trip activity & mileage system
- `daily-logs/` — Daily work logs with tasks completed, issues fixed, and context

## How It Works

This repo is the durable, versioned store of project knowledge. Claude Code reads from and writes to this repo to maintain context across sessions.
